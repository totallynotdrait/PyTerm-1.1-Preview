to run a EXE file in PyTerm follow this steps, tested on linux

1. do the command 'run cmd' will emulate your OS terminal
2. type sudo 'apt install build-essential', it will install files to run C scripts
3. then do './nameoffile'

if you want to compile a C script to a executable file the follow this steps

1. make you sure you installed the GCC compiler
2. do the command 'gcc programName.c -o programName', this command will make your C script in to a EXE file
3. to run the EXE file do './nameoffile'

source: https://vitux.com/how-to-write-and-run-a-c-program-in-linux/
